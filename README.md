# Quiz Game

A simple and interactive quiz application built using HTML, CSS, and JavaScript.

## Features

* Start screen before the quiz begins
* Multiple choice questions
* Score tracking
* Progress bar
* Result screen at the end of the quiz
* Restart quiz functionality
* Responsive and clean user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How It Works

1. The user clicks the Start Quiz button.
2. The first question is displayed.
3. The user selects an answer.
4. The application checks whether the answer is correct.
5. The score is updated.
6. The next question is displayed automatically.
7. When all questions are finished, the result screen appears.
8. The user can restart the quiz.

## Screens

### Start Screen

Displays the title of the quiz and a button to begin.

### Quiz Screen

Displays:

* Current question
* Total number of questions
* Current score
* Answer buttons
* Progress bar

### Result Screen

Displays:

* Final score
* Maximum possible score
* Restart button
